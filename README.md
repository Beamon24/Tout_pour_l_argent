# Description
Notre site « Nouveau Village Numérique » plonge les utilisateurs dans un univers éducatif et ludique, où les établissements scolaires apprennent à résister aux géants du numérique. Il promeut un **numérique inclusif, responsable et durable**, sensibilise aux logiciels libres et à la sobriété numérique, tout en expliquant les activités et enjeux liés aux GAFAM.

# Page principale
Cette page sert de page d’accueil et redirige vers les différentes fonctionnalités du site.

## Fonctionnalités
- Gestion de la musique 
- Redirection vers le snake caché
- Accès au formulaire d'inscription
- Accès au Lecteur audio rétro
- Redirection vers le Chat'Bruti (Thierry)

## Technologies
- HTML5 / CSS3  / Javascript

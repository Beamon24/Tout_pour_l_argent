# Chat'bruti
La page d'interaction avec Thierry, une IA qui prend un peu trop à coeur les questions.
Thierry utilise un prompt qui lui donne vie additionner à la requete de l'utilisateur et demande à son cerveau gemini comment répondre.

## Fonctionnalités
Génération de réponse 

## Technologies
HTML5 / CSS3 / JavaScript / API Google

## Contexte et équipe
Ce projet a été réalisé dans le cadre de la Nuit de l’Info et appartient à l’équipe Tout pour l’argent, composée d’étudiants en Licence 3 Informatique à l’Université d’Angers.
